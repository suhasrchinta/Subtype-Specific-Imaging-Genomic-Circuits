# Imaging \u2192 Gene Circuits: Interactive Demo

An interactive companion page for a Regeneron STS research submission on subtype-specific
imaging-to-gene circuits predicting pathologic complete response (pCR) to neoadjuvant
chemotherapy in breast cancer.

Follows three real patients (one per molecular subtype) through their longitudinal DCE-MRI
scans, the real gene-expression correlations those imaging features carry, and the
GSEA-derived pathway each circuit lands on \u2014 plus an interactive DCE-MRI vs. MammaPrint
AUC comparison.

**This is a public-facing demo, not part of the STS application itself** (per Regeneron STS
rules, external links are not permitted in the research report or evaluated by judges).

## Before this works: add the images

The `assets/images/{patient_id}/` folders need the PNGs generated by
`extract_demo_slices_v2.py` (the DICOM-to-PNG extraction script from earlier in this
project). For each patient, copy in:

```
T0_overlay.png
T1_overlay.png
T2_overlay.png
T3_overlay.png
```

from `~/Documents/researchlink/demo_slices/{patient_folder}/` into the matching
`assets/images/{patient_id}/` folder here. Patient ID mapping:

| `331192_HR_HER2neg` | `331192` |
| `474874_HER2pos` | `474874` |

## Running locally

No build step \u2014 plain HTML/CSS/JS. Just open `index.html` in a browser, or serve it
locally to avoid any local file-loading quirks:

```bash
cd path/to/this/repo
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Data provenance

- **Imaging features & pCR outcomes:** I-SPY2/ACRIN-6698 trial data via The Cancer Imaging
  Archive (TCIA), `datawith4visits.csv` (n=384) and `clinical_data.csv` (n=985).
- **Gene expression:** GSE194040 (n=988), Agilent gene-level expression, used to compute
  real Spearman correlations for SCUBE2, ERBB2, GRB7, FLT1, ANGPT2, and VEGFA against the
  imaging features used in each circuit's mediation test.
- **Scan images:** Real DICOM SER (signal-enhancement ratio) series and Analysis Mask
  segmentations for 3 representative pCR-responder patients, one per subtype, downloaded
  from TCIA's ISPY2 and ACRIN-6698 collections.
- **AUC and circuit statistics:** Taken directly from the accompanying research paper
  (Sections 3.1\u20133.7).

## Honesty notes baked into the content

- The TNBC circuit (BPE\u2192FLT1/ANGPT2) is presented with the caveat that BPE itself shows
  no significant genome-wide gene correlation (a real finding from the paper, Section 3.4)
  \u2014 the circuit exists at the pathway level (GSEA), not as a strong direct scatter
  relationship, and the scatter plot here honestly shows that weak correlation rather than
  hiding it.
- HR+/HER2\u2212 findings are labeled preliminary throughout, matching the paper's own framing
  (only 24 pCR-positive cases in that subtype).
- Mediation vs. simple correlation is distinguished explicitly in the UI copy.

## Not a clinical tool

This page is an educational/research companion demo. It should not be used for any
